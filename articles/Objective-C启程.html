Objective-C启程

0  评论    571  浏览

    听说Objective-C是C的超集，那么是否也就是意味着Objective-C的语法特点和C很像？要是这样的话，其实还是比较不错的。虽然大学没有写过C的代码，但是我在高中毕业的那个暑假还是接触了一段时间的C的。

    //
//  main.m
//  oc-begin
//
//  Created by tlh on 15/11/7.
//  Copyright © 2015年 zdx. All rights reserved.
//

#import <Foundation/Foundation.h>

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        // insert code here...
        NSLog(@"Hello, World!");
        NSLog(@"oc-begin");
    }
    return 0;
}

     

    这是我的第一行objective c 的代码，曾经我十分迷恋Java，因为自己在大学使用Java是比较多的，俗话说日久生情，哈哈谢谢我的Java情人陪我走这么久。但是现在可能我要移情别恋Objective-C了，大学我特别喜欢的一位老师宋铁说过，如果你对Java还是那么的感兴趣那可能你玩Java还不到位，真正好玩的是更低层的语言，不排除我在Java方面还是冰山一角，但是换一种更低层的语言我想会更有助于我在计算机的理解更佳深入，毕竟它也是一一个有艺术品位的语言。毕竟多掌握一门技能对于刚走出校门的我来说何尝不是一件好事呢？

    如果说Java是一种我认为比较有生产力的语言，那么Objective-C就是一种有艺术美感的语言，不过其实语言并不重要，就像之前蔡学镛说过做技术久了就会越来越多的关注领域和架构，语言可能并不是那么的重要，我之前是没有接触过Objective-C的，但是微信的面试官说语言并没有什么大的问题，从掌握的技能来看拿到微信的offer，我确实是非常的意外但也是情理之中的。

    那么今天就开始吧，旅程开始吧。
